About

iBetelgeuse is an open source Augmented Reality (AR) browser for iPhone, designed to be compatible with Betelgeuse, a fork of the Open Gamaray AR browser available for Android. The browser loads a virtual world–called dimension–from the web, which contains text boxes and images that are positioned in this virtual world. This virtual world is then projected on the camera image to create an augmented reality. The user can look "through" the screen of their iPhone to see this augmented world, and move and rotate the device to look around.

This browser's focus is on interaction. Any text box or image that is displayed on the screen can be clicked, which can result in a new world being loaded or a webpage being opened. This allows highly interactive worlds to be made, such as tours, quizzes, or games. Users can create their own worlds using the Dimension Description Files as described below, or using the web based Betelgeuse Designer. This designer shows labels in the virtual world and shows their corresponding images when the user gets closer. This is only one example of the application of the (i)Betelgeuse browser however. With a custom backend, much more is possible!
